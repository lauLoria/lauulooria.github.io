Control de flujo
El control de flujo se refiere a cómo se ejecutan las instrucciones en un programa de acuerdo con ciertas condiciones. Ayuda a dirigir el flujo de ejecución en diferentes direcciones:
Ejemplo: 
            
                Estas son dos estructuras muy comunes. 
                if (condición) {
                    // código a ejecutar si la condición es verdadera
                  } else {
                    // código a ejecutar si la condición es falsa
                  }

                switch (expresión) {
                    case valor1:
                      // código a ejecutar si expresión es igual a valor1
                      break;
                    case valor2:
                      // código a ejecutar si expresión es igual a valor2
                      break;
                    default:
                      // código a ejecutar si no coincide con ningún caso anterior
                  }

Manejo de errores
Se utiliza para controlar situaciones excepcionales o errores durante la ejecución de un programa. Permite capturar y gestionar estos errores de manera adecuada.
Ejemplo: 
            
                Este es un ejemplo del manejo de errores
                try {
                    // código que puede generar una excepción
                  } catch (error) {
                    console.log(error.message); // muestra el mensaje de error
                    console.log(error.name); // muestra el nombre del tipo de error
                    // otras acciones para manejar el error
                  }
Bucles e iteración
En esta podemos encontrar tres bucles:
-El bucle for se utiliza cuando se conoce de antemano la cantidad exacta de veces que se desea repetir un bloque de código.
-El bucle while se utiliza cuando no se conoce de antemano cuántas veces se desea repetir un bloque de código, pero se tiene una condición que debe cumplirse para que la repetición continúe.
-El bucle do...while es similar al bucle while, pero con una diferencia importante: el bloque de código dentro del bucle se ejecuta al menos una vez antes de verificar la condición de continuación.
Ejemplo: 
            
                Este es un ejemplo de ellos 

                for (int i = 0; i < 5; i++)
                {
                    Console.WriteLine("El valor de i es: " + i);
                }

    Números y fechas
    En Visual los objetos Number, Math y Date son herramientas que te permiten trabajar con números y fechas en JavaScript. El objeto Number se utiliza para operaciones matemáticas con números, el objeto Math proporciona funciones matemáticas avanzadas, y el objeto Date se encarga de manipular fechas y horas. Estos objetos te ayudan a realizar cálculos numéricos y manipulaciones de fechas de manera efectiva en tus programas..
    Ejemplo: 
                
                Uso del objeto Number
                let num = 3.14159;
                console.log(num.toFixed(2)); // Salida: 3.14
                console.log(Number.parseInt("10")); // Salida: 10
                
                Uso del objeto Math
                console.log(Math.sqrt(16)); // Salida: 4
                console.log(Math.max(10, 5, 8)); // Salida: 10
                
                Uso del objeto Date
                let currentDate = new Date();
                console.log(currentDate.getFullYear()); // Salida: el año actual
                console.log(currentDate.getMonth()); // Salida: el mes actual (0-11)
                console.log(currentDate.getDate()); // Salida: el día del mes actual
                console.log(currentDate.getHours()); // Salida: la hora actual

    Formateo de texto y Colecciones indexadas
    El formateo de texto implica manipular y presentar cadenas de texto en un formato específico. Las cadenas literales son secuencias de caracteres encerradas entre comillas y se usan para representar texto en un programa. Los objetos String son estructuras de datos que representan secuencias de caracteres y permiten realizar operaciones en ellas. Las colecciones indexadas, como los arreglos, almacenan elementos con índices únicos para acceder a ellos. Los arreglos tipados especifican el tipo de datos que pueden contener y brindan seguridad de tipo. Visual Studio Code es un entorno de desarrollo utilizado por programadores para escribir y depurar código en diversos lenguajes de programación

                nombre = "Juan"

    Document Object Model
    El DOM es una representación en memoria de un documento HTML que permite acceder y manipular elementos de una página web. Puedes encontrar elementos en el DOM utilizando métodos como buscar por ID, nombre de etiqueta o clase, selectores CSS, o mediante colecciones de objetos.
    Este es un Ejemplo: 
                
                  id="titulo">Hola, mundo!      
                  // Encontrar elemento por ID
                  const tituloElemento = document.getElementById('titulo');

                  // Modificar el contenido del elemento
                  tituloElemento.textContent = "¡Hola, OpenAI!";

                  // Imprimir el contenido del elemento modificado
                  console.log(tituloElemento.textContent); // Imprime: "¡Hola, OpenAI!"
